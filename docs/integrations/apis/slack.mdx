<CodeGroup>

```bash npm

```

```bash pnpm
pnpm install @trigger.dev/slack@latest
```

```bash yarn
yarn add @trigger.dev/slack@latest
```

## Authentication

Slack supports OAuth

```ts
import { Slack } from "@trigger.dev/slack";

const slack = new Slack({
  id: "slack",
});
```
